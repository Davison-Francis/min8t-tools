Free tools for email marketers

Genuinely useful, no signup, no rate limits. Each tool runs in your browser or on Cloudflare's free tier - your data never sits on a server you don't control.

UTM Campaign URL Builder

Build tracked links for your email campaigns with one click.

Email Image Compressor

Compress and resize images so your emails stay under inbox size limits.

Email Signature Generator

Outlook-safe HTML signatures with one-click copy.

Subject Line Analyzer

Score your subject line on length, spam triggers, sentiment, and more.

HTML to Plain-Text Converter

Generate the plain-text MIME alternative from any HTML email. Better deliverability, RFC 2046 compliance.

Email-Safe Font Checker

Will Roboto / Inter / Helvetica render in Outlook? Per-client support matrix and copy-paste fallback stacks.

Brand Color Extractor

Upload a logo, get a 10-step brand palette with WCAG checks. Exports to CSS / Tailwind / JSON / SCSS.

GIF Compressor

Compress animated GIFs in your browser via gifsicle WASM. Email-aware presets (Outlook-safe / mobile / max).

Spam Score Checker

Inbox Preview

CSS Inliner

MJML Converter

Compile MJML markup to email-safe HTML. Three-pane: input / output / live preview. Browser-based.

Email Header Analyzer

Trace email hops, decode SPF/DKIM/DMARC verdicts, flag suspicious patterns. RFC-5322 parser in your browser.

Bulletproof Button Generator
