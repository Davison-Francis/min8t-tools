Frequently Asked Questions

Are MiN8T's free email marketing tools really free with no signup?

Yes. Every tool on this page runs entirely in your browser or on Cloudflare's free tier. There is no signup, no rate limit, and no upload of your files to a server we control. Image compression, GIF compression, CSS inlining, MJML conversion, and inbox preview all execute client-side via WebAssembly — your data never leaves your device.

Do these email tools work in Outlook, Gmail, and Apple Mail?

The tools are specifically designed for email-marketer constraints. The CSS Inliner preserves @media queries Outlook ignores. The Image Compressor enforces sub-1MB payloads so emails render across all clients. The Inbox Preview tool simulates Gmail Web, Apple Mail iOS, and a strict-mode Outlook approximation side-by-side so you can catch cross-client failures before sending.

What is the difference between /tools and /free-tools?

min8t.com/tools is the email-marketing toolset: UTM builder, image compressor, signature generator, subject line analyzer, inbox preview, CSS inliner, MJML converter, header analyzer, button generator, A/B test calculator, and others. min8t.com/free-tools is the email-deliverability toolset: email verifier, domain health check (SPF/DKIM/DMARC), blacklist scanner, domain intelligence, DMARC analyzer, and DMARC generator. They are complementary — design with /tools, then verify deliverability with /free-tools.

Do I need to install anything to use these tools?

No installation required. Every tool runs in any modern browser (Chrome, Firefox, Safari, Edge). The heavier tools — gifsicle for GIF compression, the email-image library, the MJML compiler — load as WebAssembly binaries cached locally after first use. After the first visit, tools work even offline.

Why does MiN8T offer these tools for free?

The free tools showcase the email design and deliverability primitives that power the MiN8T platform. We use the same image compression, CSS inlining, and authentication logic in the full editor — the free tools let you try them in isolation. If they save you time, the full editor at min8t.com replaces Stripo, BeeFree, ZeroBounce, NeverBounce, Zapier, Teams, Litmus, and Mailmodo in a single subscription.

Will these tools save my data or send it to a server?

For pure client-side tools (image compressor, GIF compressor, CSS inliner, MJML converter, inbox preview, palette extractor, plain-text converter, font checker, button generator, A/B calculator) — no. The file never leaves your device. For tools that require server-side work (UTM builder shortener, spam-checker AI classifier) we use ephemeral processing — input is not logged, not stored, not sold. The tools work without an account precisely because we do not need to track you.

Can I use these tools in my own product or workflow?

The tools are designed for direct human use, not as an API. If you need programmatic access to email compression, validation, or deliverability checking, the MiN8T platform exposes a REST API and SDKs for Node.js, Python, Go, PHP, and Ruby covering the same primitives. See min8t.com/api for endpoints, rate limits, and authentication.

How often are new tools added?

New tools ship roughly every 2–4 weeks, prioritized by what email marketers ask for most often. Recent additions include the Spam Score Checker (60+ heuristic rules plus an AI classifier), the Inbox Preview tool (browser-based Gmail / Apple Mail / Outlook approximation), and the A/B Test Sample Size Calculator.
