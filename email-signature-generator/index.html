Why these templates work in Outlook

  
Outlook desktop uses Microsoft Word's HTML engine, which strips flexbox, grid, background-image, and @media queries. The templates in this tool use only what survives that engine: <table>-based layout with cellpadding/cellspacing instead of CSS margins, inline styles only (no <style> blocks, which Outlook ignores anyway), pixel-only widths, and a 500-600 px overall width that fits cleanly in mobile and desktop reading panes alike.

  
Most "free signature generator" tools online use a Vue/React form with no awareness of Outlook's quirks - paste their output into Outlook and you get a stack of bare text. The templates here have been tested against the caniemail compatibility matrix; every CSS property used is supported by at least 95% of email clients.


  
FAQ

  

    

      How do I add an HTML signature in Gmail?
      
In Gmail web, click the gear icon → See all settings → Signature → Create new. Paste the rendered signature (use the "Copy for Gmail" button - it copies as rich text, not HTML source). Save changes at the bottom of the page. Gmail's signature editor doesn't accept HTML directly; pasting rich-text content is the only path.

    

    

      How do I add an HTML signature in Outlook?
      
Outlook 2016/2019/365: File → Options → Mail → Signatures → New. The signature editor accepts pasted rich content from this tool - use "Copy for Gmail" to grab the rendered signature and paste it. For Outlook on the web, open Settings → Mail → Compose and reply, paste into the signature box. If formatting breaks, fall back to "Copy HTML source" and edit the .htm file in your Outlook signatures folder directly.

    

    

      How do I add a signature in Apple Mail?
      
Apple Mail (macOS): Mail → Settings → Signatures → click + to add. Paste the rendered signature. Apple Mail accepts rich content cleanly. To preserve exact HTML, drag the rendered HTML file from a browser tab into the signature panel - Apple Mail will keep the structure verbatim.

    

    

      Why do my email signatures break in Outlook?
      
Outlook uses Microsoft Word's HTML rendering engine, which strips most modern CSS. Specifically: no flexbox, no grid, no background-image, no @media queries, no margin (use cellpadding/cellspacing on tables instead), no rem/em (use px). The templates in this tool are built with Outlook in mind: pure table-based layouts with inline styles, fixed pixel widths, and no fancy CSS.

    

    

      Should I include a photo in my email signature?
      
Optional but signals personality. Keep it under 200 KB and at most 100×100 px (200×200 for retina). Use JPEG (universal) or PNG (if you need transparency around a circular crop). Host it on a public URL - don't paste a data: URL into the signature, since Outlook may strip it and emails on slow connections won't load it.

    

    

      What size should an email signature be?
      
Keep the total HTML under 5 KB and the photo (if any) under 200 KB. Width should be 500-600 px so it fits in mobile inboxes and doesn't overflow Gmail's reply window. Tall, decorated signatures with banner ads and quote boxes look cluttered and trigger spam-filter heuristics - keep it short, professional, scannable.

    

    

      Can I use animated GIFs in my email signature?
      
Technically yes, but most clients only show the first frame. Outlook desktop is the worst offender - it strips animation entirely. If you want motion, ship a static fallback that works as a single frame, or skip the animation.

    

    

      How do I include a calendar booking link in my signature?
      
Add the link as a regular link in the signature - your Calendly, SavvyCal, Cal.com, or HubSpot meeting URL. Use a clear call-to-action like "Book a 15-min chat" rather than the bare URL. Don't embed an iframe or button-image - both break in most clients. A plain styled link is universal.

    

    

      Do email signatures need a legal disclaimer?
      
Depends on your jurisdiction and industry. EU companies typically need company name, registered address, and registration number under e-commerce / company-law rules. Healthcare and financial services often require confidentiality notices. Check with your legal team - there's no one-size-fits-all answer.
